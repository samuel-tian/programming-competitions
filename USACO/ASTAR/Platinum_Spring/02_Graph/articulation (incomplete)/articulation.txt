Articulation Points
===================

You are given an undirected graph with N nodes and M edges. 
You should find the number of articulation points in this graph. 

PROBLEM NAME: articulation

INPUT FORMAT:

The first line contains two integers N and M (1 <= N,M <= 100,000). 
For the following M lines, ith line contains two integers a and b 
indices of the nodes.  

OUTPUT FORMAT:

Output should contain a single integer denoting the number of 
articulation points.

SAMPLE INPUT:

5 6
1 2
2 3
1 3
2 4
4 5
2 5

SAMPLE OUTPUT:

1
